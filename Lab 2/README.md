# Lab 2

**Students have to execute the following command lines:**

1. Go to the IoT repository
2. Study Lessons 1 and 2
3. Open a terminal
4. Execute command lines:
- $ hostname
- $ env
- $ ps
- $ pwd
- $ git clone https://github.com/kevinwlu/iot.git
- $ cd iot
- $ ls
- $ cd
- $ df
- $ mkdir demo
- $ cd demo
- $ nano file
- $ cat file
- $ cp file file1
- $ mv file file2
- $ rm file2
- $ clear
- $ man uname
- $ uname -a
- $ ifconfig
- $ ping localhost
- $ netstat
